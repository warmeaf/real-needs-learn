# 有了价值、共识和模式，为什么还需要求真？

> 基于梁宁《真需求》原书第四部分"求真"的深入解读

---

## 问题

价值、共识、模式已经构成了完整的商业闭环，具备了持续赚钱的能力，那为什么还有一个"求真"的环节？

---

## 核心回答

**"求真"不是价值、共识、模式之外的第四个独立环节，而是贯穿三者的一种创造者的精神状态——它回答的是：这一切是否建立在客观真实之上，还是你的主观想象？**

---

## 一、原书中的定义

原书第四部分开篇明确：**"求真——创造者的精神"**。

梁宁用《道德经》中的一句话作为纲领：

> **"弱者道之用"**

并解释：

> "强者能成事，也许是因为这人自身的资源强、能量强；**而弱者能成事，是因为他的背后有天道，他的做法符合'道'**。"

这里的"天道"，就是**真需求**。

---

## 二、为什么有价值、共识、模式还不够？

因为它们可能是建立在**主观想象**之上，而非**客观真实**之上。

### 两个典型案例对比

| 人物 | 背景 | 成就 | 核心差异 |
| ------ | ------ | ------ | ---------- |
| **王小川** | 清华学霸，奥赛金牌，搜狐副总裁 | 搜狗，后被腾讯35亿美元收购 | 强者之路：资源驱动，追逐技术浪潮 |
| **李兴平** | 初中辍学，县城网管 | hao123，一人维护5年，流量从未被搜狗超越 | 弱者之道：看到被精英忽视的真实用户 |

### 关键洞察

> "很多强者输给弱者，是因为他们的目光太集中在自己的资源和自己的目标上，看到的都是'我我我我我'。他们的光环让自己自负，**他们看不到天道**。"

李兴平看到的"天道"是什么？

> "这些弱弱的、对一切都陌生、什么都不会的互联网使用者，才是中国新网民的基本盘。"
>
> "这些弱的人，根本不会出现在那些学霸精英的视野里。"

---

## 三、求真的本质：在实然中看到应然

原书对求真的核心定义（p.428）：

> "所有的创新者，一定是不接受现实的人。同时，他们一定是行动派……**在实然的世界里看到应然，然后以自己的凡人之躯去承担神的任务，把应然变成实然，让梦想在客观世界里成为现实**。"

### 概念解析

| 概念 | 含义 | 例子 |
| ------ | ------ | ------ |
| **实然** | 现实世界的真实状态 | 县城用户不会输网址、不会打字、不知道地址栏 |
| **应然** | 你认为应该成为的样子 | 应该有一个页面，让用户点击鼠标就能上网 |
| **求真** | 从实然出发，用行动将应然变为现实 | 李兴平自学做网页，做出hao123 |

## 四、求真在商业闭环中的位置

```text
┌─────────────────────────────────────────────────────────┐
│                      求真（创造者的精神）                  │
│  ┌─────────┐      ┌─────────┐      ┌─────────┐         │
│  │  价值   │ ←——→ │  共识   │ ←——→ │  模式   │         │
│  │ (Value) │      │(Consensus)│     │ (Model) │         │
│  └────┬────┘      └────┬────┘      └────┬────┘         │
│       └────────────────┼────────────────┘              │
│                        ↓                               │
│              ┌─────────────────┐                       │
│              │   是否真实？     │                       │
│              │  · 是真实需求？  │                       │
│              │  · 还是你的想象？ │                       │
│              └─────────────────┘                       │
└─────────────────────────────────────────────────────────┘
```

### 求真对三者的检验

| 环节 | 没有求真 | 有了求真 |
| ------ | ---------- | ---------- |
| **价值** | 你"以为"用户需要 | 用户"真实"需要 |
| **共识** | 表面的认同和附和 | 真正的认同和付费 |
| **模式** | 建立在假设上的循环 | 建立在真实流转上的循环 |

## 五、李兴平的"求真"实践

### 他的用户真实状态（实然）

- 记不住复杂的英文域名
- 不会用键盘和搜索
- 不知道在哪里输入网址
- "你告诉他在地址栏输入，他会问：什么是地址栏？"

### 他的解法（应然→实然）

> "给这样的用户创造一个只要点击鼠标就可以解决所有上网诉求的场景——**不需要记住任何东西，不需要打字，不需要自己不断地做出决定**"

### hao123的追求

> "hao123不追求浏览量，不追求页面漂亮，不追求技术先进，甚至不追求广告收入。hao123追求的也许只有一件事——**'简单'**"

结果：

> "简单这个词，几乎是所有工具之争的制高点。"
>
> "网址导航站这件事，没有人可以超越hao123，是因为没有人可以比它更简单。"

---

## 六、一句话总结

**价值、共识、模式解决"怎么赚钱"，求真解决"赚的是不是真钱"——即这一切是否建立在客观真实的需求之上。**

> "互联网的天道，就是'真需求'。"

求真，就是像李兴平那样，**放下自我光环，如实看到世界本来的样子**。
